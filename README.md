# **AntiQ**

**Queues Made Smarter**

AntiQ is a web-based virtual queue management system designed for clinics, hospitals, institutes, and service centers. The platform allows users to book tokens online and track service status in real time, while enabling staff to manage service flow efficiently through an admin dashboard.

---

## 🚀 Features

### User Features

* Online token booking
* View live service status
* Real-time updates without page refresh
* Estimated waiting time
* Token history and status tracking

### Admin / Staff Features


* Secure admin login
* Live service dashboard
* Serve next, skip, pause, and resume controls
* Instant updates across all connected users

### System Features

* Real-time communication using WebSockets
* Role-based access control
* Responsive and user-friendly interface

---

* **Backend:** Node.js, Express.js

## ⚙️ How It Works

1. Users select a service and book a token online.
2. The system generates and books a token and displays live status.
3. Admin controls service flow through the dashboard.
4. Updates are synchronized instantly using WebSockets.

---

## 🎯 Purpose

AntiQ aims to reduce unnecessary waiting, improve transparency, save user's time and create a smoother service experience by digitizing traditional waiting systems.
